Program ini memerlukan beberapa perangkat lunak pendukung untuk dijalankan, yaitu:

Langkah Instalasi
- Ekstrak aplikasi web ini
- Simpan ke folder htdocs
- Di folder yang telah di ekstrak, ada satu file sql di dalam folder bernama database. Silakan impor file sql ke server web Anda.
